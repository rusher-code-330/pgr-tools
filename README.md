# PGR Tools V.beta

PGR Tools is a simple toolbox that includes calculators, text tools, password generator, timers, and YouTube download features (MP3/MP4).

---

# Features

- Calculator (addition, subtraction, multiplication, division)
- Random password generator
- ASCII text generator 
- Countdown timer
- YouTube MP3 downloader 
- YouTube MP4 downloader 
---
# launch PGR Tools
```bash
pgr
```
---

# installation (Termux / Linux)

```bash
git clone https://github.com/rusher-code-330/pgr-tools.git
cd pgr-tools
bash install.sh
cd ~
```

# installation windows

```bash
git clone https://github.com/rusher-code-330/pgr-tools.git
cd pgr-tools
install.bat
cd ~
```

# installation linux

```bash
git clone https://github.com/rusher-code-330/pgr-tools.git
cd pgr-tools
bash install2.sh
```
