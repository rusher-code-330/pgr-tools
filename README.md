# PGR Tools v2.0.1

PGR Tools is a simple cli toolbox

---

# library (in pgr tools)
```pgr
pgr install youtube
```
### youtube converter

```pgr
pgr install timer
```
### Just a timer
```pgr
pgr install ascii-text
```
---
# launch PGR Tools
```bash
pgr
```
## for update
```bash
pgr cli update
```
---

# installation (Termux / Linux)

```bash
git clone https://github.com/rusher-code-330/pgr-tools.git
cd pgr-tools
bash install.sh
cd ~
```

# installation windows

```bash
git clone https://github.com/rusher-code-330/pgr-tools.git
cd pgr-tools
install.bat
cd ~
```

# installation linux

```bash
git clone https://github.com/rusher-code-330/pgr-tools.git
cd pgr-tools
bash install2.sh
cd ~
```
